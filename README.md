# desktop-uhf-reader-csharp
C# SDK and sample code for Fongwah Desktop UHF RFID Readers. Supports tag inventory, reading/writing (EPC, TID, User), and device control via E7umf.DLL.
This SDK is provided for use with Fongwah hardware. Please refer to https://fongwahrfid.net/  for full terms and technical support.
Compatible Hardware: * Desktop Reader: U6-CU-91 , U1-CU-71
Import E7umf.DLL to your project bin folder before running.
